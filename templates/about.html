About PhishGuard AI

Intelligent phishing detection powered by deep learning

Protecting Users from Phishing Attacks with AI

PhishGuard AI uses a TensorFlow/Keras deep learning model trained on 10,000 URLs with 48 carefully engineered features to detect phishing attempts with high accuracy. Our regression model provides a continuous risk score, enabling nuanced threat assessment rather than simple binary classification.

Key Features

Deep Learning Model

TensorFlow/Keras neural network with 4 dense layers, batch normalization, and dropout for robust predictions.

JWT Authentication

Secure login system with password hashing, token-based auth, and role-based access control.

Interactive Analytics

Feature importance charts, training history, class distribution, and downloadable model assets.

Batch Prediction

Upload CSV files to analyze multiple URLs at once with detailed summary statistics.

Early Stopping

Smart training with early stopping (patience=5) to prevent overfitting and restore best weights.

Real-Time Scoring

Instant phishing risk assessment with animated gauge visualization and clear status indicators.

Help & FAQ

How do I make a prediction?

Navigate to the Predict page, enter the 48 URL features in the form, and click "Analyze Phishing Risk". You'll see a risk gauge showing the probability that the URL is a phishing attempt.

What do the risk scores mean?

Scores below 40% indicate a legitimate URL, 40-70% is suspicious, and above 70% is likely phishing. The model outputs a continuous probability, giving you nuanced risk assessment.

How do I use batch prediction?

On the Predict page, switch to the "Batch Upload" tab. Upload a CSV file with the 48 feature columns and the system will return predictions for all rows.

How is the model trained?

The model is a TensorFlow/Keras neural network trained on 10,000 labeled URLs (5,000 phishing, 5,000 legitimate) with 48 features. It uses early stopping with patience=5 to prevent overfitting, training for up to 20 epochs.
